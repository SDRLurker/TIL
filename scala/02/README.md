### Class 작성 연습

다음 object들을 class로 작성하세요. 

| object         | realName        | position      |
| -------------- | --------------- | ------------- |
| CaptainAmerica | "Steve Rogers"  | "Leader"      |
| IronMan        | " Tony Stark"   | "Range Deler" |
| Hulk           | "Bruce Banner"  | "Tanker"      |
| Thor           | " Thor Odinson" | "God"         |

| ------------- | ------------- |
